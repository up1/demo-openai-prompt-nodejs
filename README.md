# Demo Custom Prompt with NodeJS
* OpenAI
* Read data from CSV

## Processes
* Read data from csv file
* Create embedding from csv file (by cell/column in each row)
* Create prompt
* Send prompt to OpenAI

## Steps to run

### Rename file `.env.sample` to `.env` 
* Add your api key

### Install and run
```
$npm install
$node index.js 
```
